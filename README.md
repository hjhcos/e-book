# e-book
运行环境：<br>
PyCharm<br>
python3.7<br>
在`src`文件夹下输入以下命令
```python
python reapp.py
```
可以运行`reapp.py`文件<br>
在`src`文件夹里面有一个`src.zip` <br>
解压`src.zip` 文件 <br>
解压后会有一个`reapp.exe`文件<br>
可以直接双击运行<br>
```python
access.py     # 实现网站的数据爬取文件
reapp.py      # 应用程序代码
config.ini    # 初始文件
LOG.log       # 错误日志文件
```
